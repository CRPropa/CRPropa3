\doxysubsection{Detailed Description}
Photon field decorator for tabulated photon fields. 

This class reads photon field data from files; The first file must be a list of photon energies \mbox{[}J\mbox{]}, named field\+Name\+\_\+photon\+Energy.\+txt The second file must be a list of comoving photon field densities \mbox{[}1/m$^\wedge$3\mbox{]}, named field\+Name\+\_\+photon\+Density.\+txt Optionally, a third file contains redshifts, named field\+Name\+\_\+redshift.\+txt
