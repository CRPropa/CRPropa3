\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class crpropa\+::\+Vector3$<$ T $>$}
Template class for 3-\/vectors of type float, double, ... 

Allows accessing and changing the elements x, y, z directly or through the corresponding get and set methods.

Angle definitions are phi \mbox{[}-\/pi, pi\mbox{]}\+: azimuthal angle in the x-\/y plane, 0 pointing in x-\/direction theta \mbox{[}0, pi\mbox{]}\+: zenith angle towards the z axis, 0 pointing in z-\/direction
