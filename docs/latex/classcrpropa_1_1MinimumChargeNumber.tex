\doxysubsection{Detailed Description}
Deactivates the candidate below a minimum number. 

This modules deactivates the candidate below a given minimum charge number. A minimum charge number of 26 deactivates all (anti-\/) isotopes which are ranked in the periodic table before iron (Fe). In that case the property (\char`\"{}\+Deactivated\char`\"{}, module\+::description) is set.
