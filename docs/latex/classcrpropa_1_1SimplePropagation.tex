\doxysubsection{Detailed Description}
Simple rectilinear propagation in absence of magnetic fields. 

This module implements rectilinear propagation. The step size is guaranteed to be larger than min\+Step and smaller than max\+Step. It always proposes a next step size of max\+Step.
