\doxysubsection{Detailed Description}
Particle type at the source. 

This feature assigns a single particle type to the source. For multiple types, use e.\+g. \mbox{\hyperlink{classcrpropa_1_1SourceMultipleParticleTypes}{Source\+Multiple\+Particle\+Types}}. Particles are identified following the PDG numbering scheme\+: \href{https://pdg.lbl.gov/2019/reviews/rpp2019-rev-monte-carlo-numbering.pdf}{\texttt{ https\+://pdg.\+lbl.\+gov/2019/reviews/rpp2019-\/rev-\/monte-\/carlo-\/numbering.\+pdf}}
