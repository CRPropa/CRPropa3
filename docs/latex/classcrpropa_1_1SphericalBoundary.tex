\doxysubsection{Detailed Description}
Flag a particle when leaving the sphere. 

The particle is made inactive and flagged as \char`\"{}\+Rejected\char`\"{}. By default the module prevents overshooting the boundary by more than a margin of 0.\+1 kpc. This corresponds to the default minimum step size of the propagation modules (\mbox{\hyperlink{classcrpropa_1_1PropagationCK}{Propagation\+CK}} and \mbox{\hyperlink{classcrpropa_1_1SimplePropagation}{Simple\+Propagation}}).
