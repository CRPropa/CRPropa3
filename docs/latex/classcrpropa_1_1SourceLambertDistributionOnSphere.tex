\doxysubsection{Detailed Description}
Uniform random position on a sphere with isotropic Lamberts distributed directions. 

This function should be used for crosschecking the arrival distribution for a Galactic propagation with an isotropic arrival distribution at the Edge of our Galaxy. Note, that for simulation speed you should rather use the backtracking technique\+: see e.\+g. \href{http://physik.rwth-aachen.de/parsec}{\texttt{ http\+://physik.\+rwth-\/aachen.\+de/parsec}}
