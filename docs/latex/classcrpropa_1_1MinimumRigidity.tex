\doxysubsection{Detailed Description}
Deactivates the candidate below a minimum rigidity. 

This modules deactivates the candidate below a given minimum rigidity (E/Z in EeV). In that case the property (\char`\"{}\+Deactivated\char`\"{}, module\+::description) is set.
