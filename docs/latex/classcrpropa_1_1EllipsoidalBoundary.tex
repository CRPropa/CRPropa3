\doxysubsection{Detailed Description}
Flags a particle when leaving the ellipsoid. 

This module flags particles when outside of the ellipsoid, defined by two focal points and a major axis (length). The particle is made inactive and flagged as \char`\"{}\+Rejected\char`\"{}. By default the module prevents overshooting the boundary by more than a margin of 0.\+1 kpc. This corresponds to the default minimum step size of the propagation modules (\mbox{\hyperlink{classcrpropa_1_1PropagationCK}{Propagation\+CK}} and \mbox{\hyperlink{classcrpropa_1_1SimplePropagation}{Simple\+Propagation}}).
