\doxysubsection{Detailed Description}
\mbox{\hyperlink{classcrpropa_1_1Source}{Source}} distribution that follows the Galactic SNR distribution. 

The origin of the distribution is the Galactic center. The default maximum radius is set to r\+Max=20 kpc and the default maximum height is z\+Max = 5 kpc. See G. Case and D. Bhattacharya (1996) for the details of the distribution.
