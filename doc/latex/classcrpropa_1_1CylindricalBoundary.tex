Flags a particle when leaving the cylinder. This module flags particles when outside of the cylinder, defined by a radius and a height. The particle is made inactive and by default is flagged \char`\"{}\+Out\+Of\+Bounds\char`\"{}. Optionally the module can ensure the candidate does not overshoot the boundary by more than a set margin.
